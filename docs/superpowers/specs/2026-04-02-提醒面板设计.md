# 隅灵提醒面板设计（2026-04-02）

## 1. 背景
- 托盘“提醒列表”第一次点击出现闪烁并自动关闭。
- 用户希望默认只显示宠物主体，提醒管理为独立弹层，可拖拽/缩放并默认居中。
- 交互需遵循 UIUXPROMAX，美观、丝滑，并支持提醒内容展示与 CRUD。

## 2. 目标
1. 托盘触发提醒列表时稳定弹出，不再闪退，并聚焦窗口。
2. 提醒面板与宠物主体完全解耦，默认居中显示，可拖拽/缩放，关闭后重置到默认居中。
3. 面板视觉为玻璃质感、带动效，列表/表单体验细腻。
4. 代码需保留強注释、便于扩展。

## 3. 架构与流程
```
Rust handle_tray_menu_event --> emit platform:show-reminders --> Vue App.vue openReminderModal()
    |                                                                    |
    |                                            centerReminderPanel() + showReminderModal=true
    v                                                                    v
window.show() & set_focus()                        ReminderPanel component (floating layer)
```
- Rust 侧逻辑保持不变，仅确保 `window.show()` 在事件触发前调用，避免隐藏状态阻塞。
- Vue 侧 `openReminderModal()` 重置尺寸/位置并在下一帧淡入。

## 4. 组件设计
### 4.1 数据状态
- `showReminderModal: boolean`
- `reminderPanelRect: { width, height, top, left }` —— reactive，可被拖拽/缩放更新。
- `isPanelDragging: boolean`
- `resizeHandles: string[] = ["n","s","e","w","ne","nw","se","sw"]`
- `reminderError`、`reminderStore`、`petStore` 复用现有逻辑。

### 4.2 打开/关闭流程
1. `openReminderModal()`
   - `showReminderModal = true`
   - `reminderPanelRect` 重置为 360×420，随后 `centerReminderPanel()`。
   - `requestAnimationFrame` 后附加 `entering` class，触发淡入动画。
2. 关闭：按钮或点击遮罩调用 `closeReminderModal()`。
3. ESC 键监听：在 `onMounted` 内注册 `keydown`，若 `Escape` 且已打开则关闭。

### 4.3 拖拽
- 标题栏 `@pointerdown="beginPanelDrag"`。
- 记录起点坐标、起始位置，`pointermove` 更新 `reminderPanelRect`。
- `clampPanelWithinViewport()` 保证 top/left 不越界。
- 拖拽时给 `.reminder-modal` 加 `dragging` 类，降低阴影 + cursor 改为 grabbing。

### 4.4 缩放
- 四边四角追加 handle div。
- `beginPanelResize(event, direction)`：根据方向增减 width/height/left/top。
- 设定最小尺寸 320×260。
- handle 样式为 12px 半透明圆点，hover 高亮并显示对应 cursor。

### 4.5 视觉与动效
- 面板：glassmorphism，背景 `linear-gradient(180deg, rgba(15,20,35,0.92), rgba(7,10,18,0.98))`。
- 双描边 + 24px 内边距，顶部发光条 6px。
- 入场动画 250ms scale/opacity + 顶部发光条长度动画。
- 拖拽时外发光透明度 -20%。
- 列表项新增时 `transition: transform 150ms, opacity 150ms`。

### 4.6 可访问性
- 遮罩 `role="presentation"`，面板容器加 `role="dialog"` + `aria-labelledby`。
- 关闭按钮 `aria-label="关闭提醒面板"`。
- ESC 可关闭。

- 提醒列表可操作，新增/删除无 UI 异常。
- ESC/遮罩均可关闭。
